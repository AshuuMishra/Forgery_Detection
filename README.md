# Forgery Detection — Streamlit App

AI-powered document and image tampering detection using EfficientNet B4 + ELA.

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Usage

1. Open the app in your browser (default: http://localhost:8501)
2. **Sidebar → Upload Model** — upload your `full_model.pth` from Google Drive
3. **Main area → Upload Image** — upload any image to analyze
4. Click **Analyze Image**
5. View verdict, confidence score, forgery type, and heatmap

## Detects

| Type | Description |
|------|-------------|
| Copy-Move | Region copied and pasted within same document |
| Splicing | Region from another document inserted |
| Text Tamper | Text erased and rewritten |
| AI Generated | Image fully generated by AI |

## Getting Your Model

In Google Colab after training:
```python
# Download model from Drive
from google.colab import files
files.download('/content/drive/MyDrive/forgery_detection/full_model.pth')
```

## Notes

- Model runs on CPU if no GPU available (slower but works)
- Supported image formats: JPG, PNG, BMP, TIF
- Adjust Detection Threshold in sidebar (default 0.5)
  - Higher threshold = stricter (fewer false positives)
  - Lower threshold = more sensitive (catches subtle forgeries)
